# Lifelong Learning

Projeto experimental utilizando Astro, Astro DB, SSR, APIs e estudos de arquitetura frontend/fullstack.

---

# 🛠️ Stack

- Astro 6
- TypeScript
- Astro DB
- React
- MDX
- json-server
- Vercel
- Turso/libSQL

---

# 📋 Scripts principais

| Script             | Descrição                            |
| ------------------ | ------------------------------------ |
| `npm run dev`      | Inicia o servidor de desenvolvimento |
| `npm run dev:full` | Executa Astro + json-server          |
| `npm run build`    | Gera a build de produção             |
| `npm run check`    | Executa verificações do Astro        |
| `npm run sync`     | Sincroniza tipos do Astro DB         |

> Consulte [`docs/scripts.md`](./docs/scripts.md) para ver todos os scripts disponíveis do projeto.
